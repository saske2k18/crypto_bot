# Crypto Price Prediction Bot

Нейросеть для предсказания движения цен криптовалют.

## Особенности
- 📊 5 криптовалют: BTC, ETH, BNB, SOL, XRP
- ⏱️ 6 таймфреймов: 5m, 15m, 1h, 4h, 1d, 1w
- 🧠 LSTM/Transformer архитектуры
- 📈 Мульти-таймфрейм анализ

## Запуск
1. Открой в Google Colab
2. Включи GPU (Среда выполнения → GPU)
3. Запусти все ячейки по порядку

## Структура
- `data/` - загруженные данные
- `models/` - сохраненные модели
- `notebooks/` - Jupyter ноутбуки
- `src/` - исходный код
